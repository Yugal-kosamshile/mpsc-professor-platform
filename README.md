# MPSCProfessor: The Aspirant's Intelligence 🎓

A comprehensive, React-based educational platform designed to democratize competitive exam preparation for MPSC, UPSC, and State Board aspirants across Maharashtra. This platform aggregates high-yield video lectures, downloadable PDF notes, and data-driven exam analytics into a single, responsive dashboard.

## 🚀 Features

* **Interactive Video Library:** A dynamic lecture grid integrating directly with the MPSCProfessor YouTube channel, featuring category-based filtering (Geography, Polity, Science, etc.).
* **Data-Driven Analytics Dashboard:** Visual insights using Chart.js to track historical MPSC cut-off trends (PSI, STI, ASO) and subject weightage for Prelims and Mains.
* **Study Material Hub:** A dedicated section for aspirants to easily download high-quality, organized PDF notes and resources.
* **Live Notifications:** An animated news ticker to alert students of live sessions, recent uploads, and important exam updates.
* **Responsive UI/UX:** Built with a mobile-first approach ensuring a seamless experience across desktop, tablet, and mobile devices.

## 🛠️ Tech Stack

* **Frontend Framework:** React.js (Vite)
* **Styling:** Tailwind CSS / Custom CSS
* **Data Visualization:** Chart.js & react-chartjs-2
* **API Requests:** Axios (for YouTube Data API integration)
* **Icons:** SVG & Bootstrap Icons

## ⚙️ Installation & Local Setup

To get a local copy up and running, follow these simple steps:

### Prerequisites
* [Node.js](https://nodejs.org/) (v16.x or higher recommended)
* npm or yarn package manager

### Installation

1. **Clone the repository:**
   ```bash
   git clone [https://github.com/Yugal-kosamshile/mpsc-professor.git](https://github.com/Yugal-kosamshile/mpsc-professor.git)

```

2. **Navigate to the project directory:**
```bash
cd mpsc-professor

```


3. **Install dependencies:**
```bash
npm install
npm install bootstrap

```


```


5. **Start the development server:**
```bash
npm run dev

```


6. **Open your browser:**
Navigate to `http://localhost:5173` (or the port provided in your terminal).
